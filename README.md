# Smart Queue Management System

##  Description

Smart Queue Management System is a web-based application designed to reduce waiting time and improve service efficiency by managing queues digitally.
It allows users to join queues remotely and enables administrators to monitor and manage queues in real time.

---

##  Features

* Digital queue registration
* Real-time queue status tracking
* Admin dashboard for queue management
* Reduced waiting time and better user experience
* Responsive web interface

---

## Tech Stack

##  Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/ASRITHAPUCHAKAYALA/smart-queue-system.git
```

---

### 2. Run Frontend

```
cd frontend
```

 Open `index.html` using **Live Server (VS Code)**

---

### 3. Run Backend

```
cd backend
npm install
npx nodemon server.js
```

---

##  Environment Variables

Create a `.env` file inside the backend folder and add:

```
MONGO_URI=your_mongodb_connection_string
```
